
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
## Run with Docker

### 1️⃣ Build and start the containers

```
docker compose up --build
```

> The application will be available at : http://localhost:5173

### 2️⃣ Stop containers
```
docker compose down
