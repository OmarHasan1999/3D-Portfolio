# 🌌 3D Portfolio

#### A 3D interactive portfolio built with Three.js, where navigation between pages is done by moving and rotating the camera through a 3D space scene — no traditional routing, just pure 3D camera animation powered by GSAP.

## 🔗 [Live Demo](https://threejs-portfolio-50b85.web.app)

## 🚀 Project Setup

```sh
npm install
```

- ### Compile and Hot-Reload for Development

```sh
npm run dev
```

- ### Compile and Minify for Production

```sh
npm run build
```

---

## 🐳 Run with Docker

### 1️⃣ Build and start the containers

```
docker compose up --build
```

> The application will be available at : http://localhost:5173

### 2️⃣ Stop containers
```
docker compose down
```
